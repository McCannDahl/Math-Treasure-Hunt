# Clue Images

Drop your clue pictures here as:

- `clue1.png`
- `clue2.png`
- ...
- `clue10.png`

The game references them in `game.js` (see the `CLUES` array). Any
common image format works — just update the file extension in `game.js`
if you use something other than `.png`.

While images are missing the page will show a dashed orange placeholder box.
